# plugin-HomeContent
Publigin para inserir conteúdo na home principal

# Configuração básica
```
"HomeContent" => [
    "namespace" => "HomeContent",
    "config" => [
        // Título que aparece na home
        "title_home" => "", 

        // Imagens que aparece na home. Preencher como um array Ex.: ["homeContent/img/img1.png", "homeContent/img/img2.png"]
        "images_home" => ["homeContent/img/preamar1.png", "homeContent/img/preamar2.png"],

        // Texto que aparece na home
        "text_home" => "Texto que sera exibido na HOMR", 

        // Botão de ação que aparece na home. Preencher como um array Ex.: ["link do botão", "texto do botão"]
        "action_home" => ["link do botão", "texto do botão"],
    ]
]

```
